## H指数
给你一个整数数组 citations ，其中 citations[i] 表示研究者的第 i 篇论文被引用的次数。计算并返回该研究者的 h 指数。

根据维基百科上 h 指数的定义：h 代表“高引用次数” ，一名科研人员的 h 指数 是指他（她）至少发表了 h 篇论文，并且 至少 有 h 篇论文被引用次数大于等于 h 。如果 h 有多种可能的值，h 指数 是其中最大的那个。

#### 理解:
首先题目这个问的非常有意思,两个h,首先第一个h就是问你,一篇论文是否满足大于h?第二个h是问你像这样的个数是否大于等于h个?
***
这你能想到什么?在日常生活中,似乎也有一些这样的问题是这么问的,比如说在一个班级里面,每个人都有h个苹果,问你是否这个班级大于h这个人,有没有这个味道?换句话说,在一个长方形里面,当边长为一,就是他的纵向微元,横向为边长h,问你是否存在一个这样的长大于h边?
***
***
那么换句话说我们是否可以看成面积,是否存在一个满足大于h*h的面积的矩形呢?
由此可以推导出我们最精妙的解法
创建一个木桶,将这个塞入进去;
如果说,对于这个木桶,我们作为一个限定调节,也就是对数据进行一次有归纳的整理.比如我们假定这个木桶是conut,一个数组形式,我们由于一个最简单基础的问题,假如说被引用次数是a,那么满足a的有多少个?这个就是用一个简单的木数组来装载;

#### 解法一:
```java

class Solution {
    public int hIndex(int[] citations) {

        int n = citations.length;
        // 创建一个桶，长度为 n+1，索引代表引用次数，值代表有几篇论文
        int[] count = new int[n + 1];
        
        // 1. 把论文扔进对应的桶里
        for (int c : citations) {
            if (c >= n) {
                count[n]++; // 引用量超过 n 的，全算在 n 头上
            } else {
                count[c]++; // 引用量为 c 的，放进第 c 个桶
            }
        }
        
        // 2. 从后往前累加论文数量，寻找 H 指数
        int totalPapers = 0;
        for (int i = n; i >= 0; i--) {
            totalPapers += count[i];
            // 如果当前累加的“高引用论文总数” >= 当前设定的 H 指数门槛 i
            if (totalPapers >= i) {
                return i;
            }
        }
        
        return 0;
    }
}
```

#### 感悟:
    首先,这个题最巧妙的是类似于面积一样的涉及,这里我们总共有两个解法
    第一个解法就是非常常规的先给数组排序,然后想象一下这个数组是一个条形统计图,那么h从0开始循环比对,我们根据h是从末尾最长的开始,这样一方面可以限制h的增长,另外一方面就是限制h是否合适.如果不合适,就会出现一个这样的情况是什么,就是h会顺势成章的大于循环遍历到当前的山峰,他会小于这个山峰
    概率是多少呢?是必定小于!! 


#### 解法二:
```java
class Solution {
    public int hIndex(int[] citations) {

        //记录最高的那根柱子他的下标,让他为一,我们假设最高的主子他为h,然后往后推,第二高的为2,这样一定是满足的,一直到最后呢

         int h = 0;
        Arrays.sort(citations);
        for(int i =citations.length-1;i>=0;i--){
            // 这个citations[i]>h必须,所以最大的开始先
            // 我们先记录这下表伪h
            int counth = citations.length -i;
            if(citations[i]>=counth){
                h++;

            }
            if(citations[i]<counth){
                return h;
            }
            
        }
        return h;
        

        
    }
}
